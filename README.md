# Ember & Ash — Restaurant Booking

A refined restaurant table booking application built with Next.js, featuring a multi-step reservation flow with experience selection, availability checking, and booking confirmation.

![Ember & Ash](restaurant_view.png)

## Features

- **Experience Selection** — Choose from four distinct dining experiences: The Hearth, The Chef's Table, The Courtyard, and The Private Cellar
- **Smart Availability** — Find available time slots based on party size, date, and preferred time
- **Multi-Step Booking Flow** — Guided wizard from experience selection through to confirmation
- **Dark Theme** — Elegant dark UI with amber/gold accents and frosted glass effects

## Tech Stack

- **Next.js 16** with React 19 and TypeScript
- **Tailwind CSS 4** for styling
- **Playfair Display & Roboto** fonts

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3021](http://localhost:3021) to view the app.

## Build

```bash
npm run build
npm start
```
